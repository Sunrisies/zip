# Sunrise Zip v1.0.1 Release Notes

## 概述

Sunrise Zip v1.0.1 是一个重要版本更新，带来了完全重写的压缩引擎和多项性能优化。这个版本专注于提升压缩速度、降低资源占用，并提供更好的用户体验。

## 主要新功能

### 1. 全新的压缩引擎

- 完全重写的压缩核心，支持多种压缩算法
- 智能文件类型识别，根据文件扩展名自动选择最佳压缩方法
- 支持的压缩方法：Stored、Deflated、DeflatedZlib、DeflatedZlibNg、Bzip2、Zstd

### 2. 多线程并行压缩

- 实现高效的多线程压缩管道
- 使用生产者-消费者模式处理压缩任务
- 可配置线程数，充分利用多核处理器资源

### 3. 大文件优化处理

- 针对大文件实现内存映射技术
- 显著减少内存占用，避免大文件压缩时的内存溢出
- 优化文件读取策略，小文件直接读取，大文件使用内存映射

### 4. 智能压缩策略

- 根据文件类型自动决定是否压缩
- 对不适合压缩的文件类型（如PDF、MP3、MP4等）直接存储
- 对适合压缩的文件类型（如TXT、JSON、代码文件等）应用压缩

### 5. 彩色日志系统

- 实现彩色控制台输出，提升日志可读性
- 支持文件滚动日志，自动管理日志文件大小
- 不同级别日志使用不同颜色区分（警告黄色、信息绿色等）

### 6. 进度显示和统计

- 实时显示压缩进度
- 提供详细的压缩统计信息
- 显示原始大小、压缩后大小和压缩比
- 显示压缩耗时等性能指标

### 7. 文件完整性验证

- 压缩完成后可验证压缩文件完整性
- 显示压缩文件中的文件列表和详细信息
- 计算文件校验和，确保压缩过程无误

### 8. 性能优化

- 优化Rust编译配置，启用最高优化级别
- 启用链接时优化(LTO)，减少二进制大小
- 使用单个代码生成单元，提升运行效率
- 自动剥离调试信息，减小文件体积

## 命令行参数更新

新增参数：

- `-d, --detail`: 显示压缩详情和统计信息
- `-b, --verbose`: 显示验证信息

## 兼容性说明

- 支持Windows
- 兼容主流的ZIP解压工具

## 已知问题

- 在处理极大量小文件时，内存占用可能较高
- 部分特殊文件名可能导致压缩失败
